# SignalrApp

# WebApi
Create Dotnet App

dotnet new webapi

Add SignalR package 
dotnet add package Microsoft.AspNetCore.SignalR.Client

## Run Api
dotnet run

# UI 

Create React App
npx create-react-app my-app --template typescript

Add SignalR package
npm install @microsoft/signalr --save

## Run UI

cd Signalr.Ui 
npm install 
npm start
